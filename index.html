Function Overview

The script includes two main functions:

  1. Get-RandomQuery: Generates a random search query using a list of words.
  2. Perform-GoogleSearch: Executes a Google search with the generated query and retrieves URLs.

Function: Get-RandomQuery

This function takes a string of words, splits them into individual elements, selects two at random, and forms a search query. It is designed to introduce variety into automated searches.

# Function to generate a random search query from a list of comma-separated words
function Get-RandomQuery {
    param (
        [string]$wordBase # Input: A single string of words separated by commas, representing possible search terms.
    )

    # Splits the input string into an array of individual words using commas as separators.
    $words = $wordBase -split ',' 

    # Selects two random words from the array and combines them into one search query string.
    $randomQuery = ($words | Get-Random) + " " + ($words | Get-Random) 

    # Outputs the generated search query to the console, providing visual feedback of what’s happening.
    Write-Output "Generated random query: $randomQuery" 

    # Returns the generated query back to the main function or process that called this function.
    return $randomQuery 
}

Function: Perform-GoogleSearch

This function uses the previously generated query to perform a Google search, capturing and filtering URLs from the results.

# Function to perform a Google search using the generated query and retrieve URLs from the search results.
function Perform-GoogleSearch {
    param (
        [string]$query # Input: A search query generated by Get-RandomQuery, e.g., "music irrigation".
    )

    # Constructs the Google search URL with the query string included.
    $searchUrl = "https://www.google.com/search?q=$query" 

    # Sends a web request to the constructed URL to fetch the search results.
    $response = Invoke-WebRequest -Uri $searchUrl -UseBasicParsing -ErrorAction Stop 

    # Extracts all the links from the search results that start with 'http', filtering out non-relevant links.
    $links = $response.Links | Where-Object { $_.href -match '^http' } | Select-Object -ExpandProperty href 

    # Outputs a confirmation message that the search was performed successfully.
    Write-Output "Search performed successfully with query: $query" 

    # Returns the array of URLs gathered from the search results.
    return $links 
}
